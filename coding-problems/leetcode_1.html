This problem comes from Leetcode but I found the Leetcode link as the "easiest" in Neetcode's section on Arrays and hashing. You can find it on Leetcode here. Here's the prompt:

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

So, I had done this one before. I didn't look at the solution in the editor this time around and just deleted and redid it. This is what I came up with:

        
          class Solution:
              def twoSum(self, nums: List[int], target: int) -> List[int]:
                  hashnums = {}
                  for ind, n in enumerate(nums):
                      diff = target - n
                      if diff in hashnums:
                          return [ind, hashnums[diff]]
                      hashnums[n] = ind
                  return False

I think this is the standard solution you'll find by googling this problem online. Fairly standard and straightforward list problem. I'll go ahead and say, it's O(n) time complexity and O(n) space complexity, for hashnums.

I want to drag myself online though. Leetcode lets you look through your historical submissions. This is what I found when I did that:

        
            class Solution:
                def twoSum(self, nums: List[int], target: int) -> List[int]:
                    hashnums = {ind: n for ind, n in enumerate(nums)}
                    for i in range(len(nums)):
                        r = target - nums[i]
                        #print(i)
                       # print(r)
                        #print(target)
                        #print(nums[i])
                        if r in hashnums.values():
                            indices = list(filter(lambda item: item[1] == r and item[0] != i, hashnums.items()))
                            #print(indices)
                            if indices:
                                return [i, indices[0][0]]
